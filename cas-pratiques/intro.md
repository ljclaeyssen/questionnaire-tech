# Cas Pratiques

Exercices de live coding pour entretien Angular. Format : enonce court, criteres d'evaluation, indices progressifs.

## Junior (30 min)

- **Calculatrice** - Event binding, gestion d'etat local
- **Todo List** - @for, @if, gestion de liste
- **Convertisseur d'Unites** - Two-way binding, formulaire simple
- **Recherche Autocomplete** - RxJS (debounceTime), async pipe

## Confirme (45 min)

- **Formulaire d'Inscription** - Reactive Forms, validators custom, cross-field
- **Leaderboard du Zoo** - Routing, services, state management
- **Formulaire Multi-etapes** - Stepper, FormGroup compose, validation progressive
- **Configurateur de Produit** - RxJS combineLatest, calculs reactifs
- **Editeur avec Guard** - CanDeactivate fonctionnel, dirty state

## Senior (60 min)

- **Dashboard Analytics** - Signals, @defer, state management, smart/dumb components
- **Refactoring Legacy** - Migration NgModule vers standalone, *ngFor vers @for, @Input vers signals
